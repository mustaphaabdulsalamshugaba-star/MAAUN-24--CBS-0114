# Mini CBT Web App

This is a simple Computer-Based Test (CBT) application built using Flask.

## Features
- Displays multiple questions
- Tracks user score
- Shows result with timestamp
- Uses Stack (LIFO) for storing answers

## How to Run

1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
